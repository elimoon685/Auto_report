# 宏观流动性与利率跟踪系统

## 一、项目简介

本项目构建了一套结构化的宏观与利率跟踪框架，
围绕以下传导链条展开分析：

流动性 → 银行负债 → 利率曲线 → 信用扩张 → 宏观景气

通过整合货币市场高频数据、同业存单利差、
国债收益率曲线以及月度宏观指标（社融、PMI、CPI、PPI），
生成系统化的 LaTeX 固收研究报告。

项目目标是模拟资产管理机构或卖方固收研究部门的工作流程，
建立一套可持续迭代的宏观利率分析体系。

---

## 二、核心分析框架

本系统分为五个层级：

### 1️⃣ 资金面层
- DR001
- DR007
- OMO 7天净投放
- MLF 净投放

目的：
识别短期流动性压力及政策对冲行为。

---

### 2️⃣ 银行负债层
- NCD AAA（3M / 6M / 1Y）
- NCD – DR007 利差

目的：
观察短端资金压力是否向银行中期负债成本传导。

---

### 3️⃣ 利率曲线层
- 10Y – 1Y 利差
- 10Y – 3M 利差
- 收益率曲线快照
- 10Y vs DR007

目的：
分析曲线形态变化及长端定价逻辑。

---

### 4️⃣ 信用扩张层
- 社会融资规模
- 人民币贷款
- 票据融资
- 企业债券发行

目的：
评估信用扩张结构及可持续性。

---

### 5️⃣ 宏观景气层
- PMI
- CPI
- PPI

目的：
判断信用扩张是否向实体经济传导。

## 五、研究逻辑说明

本系统强调“结构传导”而非单一指标判断：

1. 资金是否阶段性紧张？
2. 政策是否进行对冲？
3. 负债成本是否上移？
4. 曲线是否发生结构变化？
5. 信用扩张是否转化为景气改善？

通过多层级交叉验证，
避免单指标误判市场方向。

---

## 六、阶段样本（2025年12月 – 2026年1月）

阶段特征：

- 跨年资金扰动集中在7天期限
- OMO净投放对冲短端冲击
- NCD利差短暂收窄后修复
- 曲线阶段性走陡后回归平缓
- 1月社融显著放量
- PMI回落至荣枯线以下
- 通胀水平维持低位

阶段判断：

短期流动性扰动 + 政策对冲修复 + 信用阶段性放量，
但实体景气改善仍需验证。
